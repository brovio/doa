# Design System

## Overview
This directory contains all design system related components, guidelines, and decisions.

## Structure
- `/tokens` - Design tokens (colors, typography, spacing, etc.)
- `/components` - Design system components
- `/patterns` - UI patterns and guidelines
- `/decisions` - Architecture and design decisions
- `/utils` - Utility functions and helpers

## Memory Bank
The Memory Bank stores important design decisions and rationale to ensure consistency and provide context for future development.

## Usage
All components and styles should be built using the design system as the single source of truth.
