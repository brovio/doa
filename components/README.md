# Shared Components

## Overview
This directory contains reusable components that are shared across different parts of the application.

## Component Categories

### Layout Components
- Header
- Footer
- Sidebar
- Container
- Grid system

### Navigation Components
- Navigation bar
- Breadcrumbs
- Pagination
- Tabs

### Form Components
- Input fields
- Buttons
- Select dropdowns
- Checkboxes and radio buttons
- Form validation

### Data Display Components
- Tables
- Cards
- Lists
- Badges
- Progress indicators

### Feedback Components
- Alerts
- Toast notifications
- Modals
- Spinners

### Utility Components
- Icons
- Avatars
- Tooltips
- Popovers

## Memory Bank Entry
All shared components should follow the design system guidelines and be documented with usage examples and props specifications.
