Memorial Editor Component

A calm, step-based memorial editor with autosave, summary step, and clear editing flow

Step-Based Flow

Organized into clear steps: Biography, Dates, Gallery, Timeline, Key Memories, Privacy, and Summary.

Auto-Save

Automatically saves your progress every 30 seconds to prevent data loss.

Responsive Design

Works beautifully on all devices from desktop to mobile.

How to Use

  1. Import the MemorialEditor component from the components directory
  2. Pass onSave and onCancel callback functions
  3. Optionally provide initial data for editing existing memorials
  4. The component handles all state management and user interactions
